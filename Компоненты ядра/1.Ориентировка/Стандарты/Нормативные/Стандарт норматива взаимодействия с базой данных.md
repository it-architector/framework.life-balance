## Стандарт норматива взаимодействия с базой данных

> Обозначения взяты из стандарта модуля базы данных mysql

### Норматив взаимодействия с информацией

```php
    'Добавление' => [...],
    'Количество' => [...],
    'Получение'  => [...],
    'Изменение'  => [...],
    'Удаление'   => [...]
```

<hr>

### Добавление информации
```php
    'Добавление' => array(
    
            {ОБОЗНАЧЕНИЕ_ДЕЙСТВИЯ} => array(
                'Таблица'   => {ОБОЗНАЧЕНИЕ_ТАБЛИЦЫ},
                'Передачи'  => {СПИСОК_ПЕРЕДАЧ}
                ),
            {ОБОЗНАЧЕНИЕ_ДЕЙСТВИЯ} => array(
                'Таблица'   => {ОБОЗНАЧЕНИЕ_ТАБЛИЦЫ},
                'Передачи'  => {СПИСОК_ПЕРЕДАЧ}
                ),
            ...
            
        )
```
